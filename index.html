How You Will Update It

Manage DP & Projects

This website is set up so that only the person with access to the GitHub repository can change the content.

Change DP

Update your profile photo

Replace the image file and change one line in the data file.

  1. Upload your DP file into the website folder.
  2. Best place: assets/profile.jpg.
  3. Open data.js.
  4. Change the dp value to your file path.
  5. Commit the change on GitHub.
Add Projects

Update gallery with photo + description + price

Each new project is just one new image and one new object in the projects array.

  1. Upload the project photo to assets/projects/.
  2. Open data.js.
  3. Add a new project object inside projects.
  4. Write title, description, price, category, and image path.
  5. Commit changes and the website updates automatically.
